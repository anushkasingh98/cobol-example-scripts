# Samples/IC202A.CBL

### File Handling
- Selects PRINT-FILE assigned to "report.log"

### Data Structures
- PRINT-REC: X(120)
- DUMMY-RECORD: X(120)
- WS1: S999
- WS2: S999 COMPUTATIONAL
- DN1, DN3: S99
- DN2, DN4: S99 COMPUTATIONAL

### Inputs / Outputs
- Inputs: DN1, DN2, DN3, DN4 (USING clause)
- Outputs: DN3, DN4 (modified in procedure)

### Main Procedure
- Moves DN1 to WS1
- Increments WS1 and WS2
- Moves WS1 to DN3
- Moves WS2 to DN4
- Exits program
