# Samples/IC215A.CBL

### Data Structures
- DN2: Signed 1-digit numeric
- DN3: Signed 1-digit numeric

### Inputs / Outputs
- Input: DN2, DN3 (via LINKAGE SECTION)

### Main Procedure
- CALL-TEST-2: Sets DN2 to 1
- CANCEL-TEST-1:
  - Cancels program "IC214A"
  - Sets DN3 to 1
- IC215-EXIT: Exits program
